# Observatório e Laboratório de Produções Multissemióticas
## Political Remix: Sampling e Remix entre Diferentes Humores

### Uma Análise Dialógica e Enunciativo-Discursiva do Discurso Político Digital

---

## 🎯 Contexto e Motivação

Este repositório materializa um projeto de pesquisa que se insere na tradição bakhtiniana de análise do discurso, compreendendo as produções midiáticas como enunciados concretos situados em contextos sócio-históricos específicos. Partindo de uma perspectiva crítica e dialógica, investigamos como o discurso político se constrói e se transforma através das práticas de *sampling* e *remixagem* no ambiente digital.

### Por que criar um observatório?

Em tempos de proliferação de discursos políticos nas plataformas digitais, torna-se imperativo desenvolver ferramentas analíticas que nos permitam compreender não apenas o **que** se diz, mas **como** se diz, **quando** se diz e em **quais contextos** esses enunciados ganham vida. Nossa abordagem bakhtiniana reconhece que todo enunciado é fundamentalmente dialógico, estabelecendo relações responsivas com outros enunciados, outros discursos, outras vozes sociais.

### O que já existe por aí?

Embora existam diversas ferramentas de análise de conteúdo digital, poucos projetos abordam sistematicamente a dimensão **enunciativo-discursiva** das produções políticas em plataformas como o YouTube. Nossa contribuição reside em:

1. **Perspectiva teórica integrada**: Combinação da teoria bakhtiniana com métodos computacionais
2. **Foco na multissemiótica**: Análise não apenas do verbal, mas das relações entre diferentes linguagens
3. **Dimensão crítica**: Compreensão dos discursos em suas determinações sociais e ideológicas
4. **Metodologia aberta**: Disponibilização de código, dados e processos para a comunidade acadêmica

## 🔍 Nosso Foco

### O que queremos observar?

Nosso objeto de estudo são as **lives semanais** transmitidas no canal oficial de Jair Bolsonaro no YouTube durante seu mandato presidencial (2019-2022). Focamos especificamente em:

- **Estratégias discursivas** relacionadas à pandemia de COVID-19
- **Construção de ethos** presidencial em ambiente digital
- **Relações dialógicas** com outros discursos (científico, midiático, popular)
- **Processos de sampling e remixagem** como práticas de resistência e crítica

### Quais são nossas motivações?

1. **Pedagógica**: Desenvolver metodologias de ensino-aprendizagem que capacitem estudantes para análise crítica de discursos digitais
2. **Científica**: Contribuir para os estudos do discurso político em ambiente digital
3. **Social**: Promover letramento crítico em mídia digital
4. **Ética**: Fomentar práticas responsáveis de análise e produção de conteúdo político

### Por meio de quais recursos tornaremos as informações públicas?

- **Base de dados aberta** com metadados das lives analisadas
- **Notebooks Jupyter** documentando todo o processo de análise
- **Visualizações interativas** dos dados coletados
- **Produtos multissemióticos** (supercuts, infográficos, análises)
- **Documentação metodológica** detalhada

## 📊 Da Observação ao Público

### Nossos Dados

O corpus principal consiste em:
- **Lives semanais** do canal oficial (2019-2022)
- **Legendas automáticas** extraídas via YouTube API
- **Metadados** de publicação e engajamento
- **Segmentos temáticos** identificados através de palavras-chave

### Que informações podemos levantar?

- Frequência e contextos de uso de termos relacionados à saúde pública
- Estratégias de construção de autoridade discursiva
- Padrões temporais de abordagem de temas específicos
- Relações intertextuais e interdiscursivas

### Onde buscar informações e dados?

1. **YouTube API v3** para metadados e legendas
2. **Arquivo de transmissões** do canal oficial
3. **Bases de dados públicas** sobre políticas de saúde
4. **Corpus de referência** de discursos científicos sobre COVID-19

### Como levantar os dados?

Nossa metodologia segue um fluxo estruturado em 6 etapas, documentado nos notebooks:

1. **Settings** (`1_Settings.ipynb`): Configuração do ambiente e APIs
2. **Lives da Semana** (`2_LivesDaSemana.ipynb`): Identificação e filtragem do corpus
3. **Metadados** (`3_Metadados.ipynb`): Coleta de informações contextuais
4. **Subtitle** (`4_Subtitle.ipynb`): Processamento de legendas
5. **Sampling** (`5_Sampling.ipynb`): Identificação de segmentos temáticos
6. **SuperCut** (`6_SuperCut.ipynb`): Produção de montagens críticas

## 🚀 Como Usar Este Repositório

### Pré-requisitos

- Python 3.8+
- Jupyter Notebook
- Chave da YouTube API v3
- Bibliotecas: pandas, matplotlib, webvtt, googleapiclient

### Instalação

```bash
git clone https://github.com/dumoura/supercut-capitao-cloroquina.git
cd supercut-capitao-cloroquina
pip install -r requirements.txt
```

### Configuração

1. Obtenha uma chave da YouTube API v3
2. Configure a chave no notebook `1_Settings.ipynb`
3. Execute os notebooks sequencialmente
